#admin-project项目介绍
1. 项目所用技术栈
    * springboot
    * thymeleaf
    * mybatis
    * ......
2. 项目架构<br>
    这是一个前后端不分离的项目，后端采用的是springboot技术栈，持久层由mybatis处理，搭配缓存服务器redis记录session来提高项目的性能
    前端使用的是bootstrap框架配合thymeleaf模板来展示前台内容
3. 项目介绍<br>
    该项目目的是为了结合更多的spring生态的技术栈，通过实操来掌握springboot的基本使用
    以及更深层次的对spring生态的理解和掌握
